# ⚔️ StockDuel

Real-time market duels.  
Pick a side. Beat the market.

---

## 🚀 What is StockDuel?

StockDuel is a fast-paced trading game where users compete by joining either side of a market:

- 🟢 UP TEAM
- 🔴 DOWN TEAM

Each round is time-based, creating intense, short battles driven by market movement.

---

## ⚡ Features

- ⚔️ Live duel system (UP vs DOWN)
- ⏱️ Time-based rounds
- 📊 Real-time price simulation
- 💰 Dynamic pools
- 🐋 Whale alerts
- 🔥 Win streak system

---

## 🧠 Concept

Unlike traditional prediction markets, StockDuel focuses on:

- Speed over analysis  
- Emotion over logic  
- Competition over speculation  

This transforms trading into a game.
