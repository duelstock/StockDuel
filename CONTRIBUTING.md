Contributing

Thank you for your interest in contributing to StockDuel.

---

Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch

git checkout -b feature/your-feature-name

---

Development

- Keep changes minimal and focused
- Do not break existing UI/UX
- Maintain clean and readable code

---

Pull Requests

Before submitting a PR:

- Ensure the app still runs correctly
- Avoid unnecessary refactors
- Clearly describe your changes

---

Guidelines

- UI consistency is important
- Performance matters
- Simplicity is preferred over complexity

---

Ideas for Contribution

- UI improvements
- Animation enhancements
- Market simulation logic
- Wallet integration
- Leaderboard system

---

Note

This project is currently a prototype.
Major architectural changes may happen.

---

Thanks for contributing.
